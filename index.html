Open Source Software Foundation


FOSS MEC is a non-profit foundation that aims at promoting and strengthening the Free and Open Source Software (FOSS) ecosystem among MECians.

About

FOSS MEC (Free and Open Source Software at MEC) is a student-driven community that embraces the culture of openness, collaboration, and innovation.

The term open source refers to something people can modify and share because its design is publicly accessible. Open source software is software with source code that anyone can inspect, modify, and enhance.People prefer open source software because they have more control over that kind of software. They can examine the code to make sure it's not doing anything they don't want it to do, and they can change parts of it.Open source powers the most critical organizations and systems. Many militaries and defense agencies rely on Linux for control and infrastructure.

We aim to empower students by encouraging contributions to open-source projects, conducting workshops, and fostering a community built on sharing knowledge freely.
